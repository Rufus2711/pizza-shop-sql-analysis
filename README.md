# Pizza Sales SQL Analysis 



This repository contains a SQL analysis project based on a pizza sales dataset.  

Queries are organized into \*\*Basic → Intermediate → Advanced\*\* levels to demonstrate practical SQL skills used in analytics.



---



\##  Project Objective



Analyze pizza sales data to answer business questions like:

\- Which pizza types/categories generate the most revenue?

\- What are the top-selling pizzas?

\- Revenue share (%) by pizza type

\- Top 3 pizza types per category using ranking/window functions
